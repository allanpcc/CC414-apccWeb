Asegurar los endpoints del API Gateway de la tarea pasada

La seguridad de los endpoints see realizo por medio de API Keys. Para poder acceder a los metodos del API es necesario mandar el API Key como header, si el header no coincide con el API Key establecido para el API, entonces no se tendra acceso.
A continuacion se muestra el ejemplo de como debera ser enviado el header

headers: {'x-api-key': 'API KEY AQUI'}

La llave de acceso se genera desde el mismo servicio de API Gateway
